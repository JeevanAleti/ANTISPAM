Anti Spam-Spam Blocker For Blogs


The purpose of this document is to define scope and requirement of spam blocker for blogs for a corporate house,which to have control over comments posted by readers on their blogs.since the blogs hosted on internet, it was extremely critical to prevent unacceptable content in comments.

The proposed system will provide an effective way to determine the quality of comment in the blog article, whenever a reader adds a comment .if the comment entered has unacceptable words the system will not publish the comments instantly; instead it will mark the comment for moderation by the authorized user

This document is the primary input to the development team to architect a solution for this project
